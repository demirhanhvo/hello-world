# hello-world
Öncelikle değişkenlerimi atayıp herhangi bir hata durumunda verilecek uyarıları printf komutuyla yazdırmakla başladım. Fren durumu, kapı durumu, batarya sıcaklığı ve şarj yüzdesi değişkenleri için sınır değerlerini "if" ile koşulun sağlandığı durumda uyarıları vermesini sağladım. 

Sonrasında, değişkenlere kodun en başında değer girerek bunların çalışıp çalışmadığını kontrol ettim. Sonrasındaki adımda değerleri kullanıcının girebileceği şekilde güncelledim. 

Son olarak, kullanıcının girdiği değerlerde bir mantıksızlık olması durumunda verilecek uyarıları ayarladım. Edge cases durumlarında program kendini durduruyor. 

Not: Batarya sıcaklığının yüksek olması, kendi adıma önemli bir sorun olduğu için eğer batarya sıcaklığı 60'tan yüksekse bunu direkt edge case olarak değerlendirdim.
